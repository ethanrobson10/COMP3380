# COMP 3380 Project
## NHL Database

## Running Program
1. Navigate to the interface directory:
  ```bash
    cd interface
  ``` 

2. Run the program:
  ```bash
    make run
  ``` 

## Populating Database
- The database has already been populated by executing the populate.SQL file
- Repopulating database is possible through the main interface by typing the 'repop' command
